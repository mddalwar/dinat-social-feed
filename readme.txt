If you want to show Instagram Feed in your WordPress website, you can use this. You can use this without any configure. It's very easy to use.

== Description ==
Any one can use Dinat Social Feed for showing instagram photos in their wordpress website. You don't need to make payment any amount for using Dinat Social Feed. So let's enjoy.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dinat-social-feed` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Appearance > Widgets. 
4. You will be find a new widget which name is "Dinat Social Feed".
5. Just put Dinat Social Feed in your any sidebar area.
6. Enjoy.


== Frequently Asked Questions ==

= Need any configuration for this? =

No, You don't need any configure for this plugin.

= How can I change total photo limit? =

You will be able to change total photo limit when you will be place Dinat Social Feed widget in any sidebar area.
